# my_threejs_gui

---

To use this repo:

## 0. Clone this repo
```sh
git clone https://github.com/bhomaidan1990/my_threejs_gui_example.git
```

## 1. Install necessary Packages:
```sh
npm i
```

## 2. Build
```sh
npm run build
```

## 3. Run the server
```sh
npm run serve
```

## 4. Open the browser to the following ip/port
> [localhost](http://127.0.0.1:3000)
